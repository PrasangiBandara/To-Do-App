# To-Do App

<p>This is a simple To-Do Application implemented using TypeScript, HTML, and CSS. Users can use the application to Add, Delete, and Edit tasks, as well as mark them as Complete or Incomplete.</p>


<h3>Features</h3>

<ul>
- Create a new task by entering the task name and clicking the "Add" button.<br>
- Remove a task by clicking the "Delete" button next to the task.<br>
- Edit a task by clicking the "Edit" button next to it.<br>
- Mark a task as complete or incomplete by checking or unchecking the checkbox next to the task.<br>
- Show the Total number of Tasks as well as the number of Completed Tasks and Incompleted Tasks seperately.<br>
</ul>


<h3>Usage</h3>
<ul>
- Simply open the index.html file in a web browser to utilize the app.<br>
- To create a new task, enter the Task name into the input area and press the "Add" button.<br> 
- Click the "Delete" button next to the task to remove the task. <br>
- To edit a task, click the "Edit" button next to it and make the modifications you need. <br>
- Check or uncheck the checkbox infront of the task, to mark it as complete or incomplete. <br>
- The Total number of Tasks, number of Completed Tasks and the number of Incompleted Tasks are displayed at the bottom of the task page. <br>
</ul>
